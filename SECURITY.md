# Security Policy

## Supported versions

Only the latest published version of `@mradex77/google-play-scraper` receives security updates.

## Reporting a vulnerability

Please do not open a public issue for security problems. Instead, report them privately through [GitHub security advisories](https://github.com/MrAdex77/google-play-scraper/security/advisories/new). You will receive a response within a few days. Once the issue is confirmed and fixed, the fix is released and the advisory is published.

## Scope

This library performs outbound HTTP requests to play.google.com and parses the responses. Reports about unsafe parsing of remote content, dependency vulnerabilities, or ways a malicious response could execute code are all in scope.
